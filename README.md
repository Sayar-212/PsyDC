# PsyDC - Psychological Data Collection & Validation Platform

A clinical validation platform for reviewing AI-generated psychological assessment data, specifically focused on PHQ-9 (Patient Health Questionnaire-9) depression screening responses.

## ğŸ¯ Overview

PsyDC enables clinicians to validate synthetic psychological assessment data generated by AI. The platform presents patient responses to PHQ-9 questionnaires along with AI-calculated scores, allowing clinicians to verify accuracy and provide corrections.

- **Clinician Authentication**: Consent-based entry with clinician name tracking
- **Data Validation Interface**: Review 350+ synthetic patient responses
- **Real-time Scoring**: PHQ-9 score calculation with severity classification
- **Validation Workflow**: Mark assessments as valid/invalid with optional corrections
- **Progress Tracking**: Visual progress indicators and statistics
- **Data Export**: PDF report generation with validation results

## ğŸ® Usage

### For Clinicians

1. **Start**: Open `psy_data_collection.html` in a browser
2. **Read Materials**: Click "Read Details" and "Guidelines" (required)
3. **Begin Survey**: Click "Let's PsyDC" button
4. **Enter Information**: Provide name and consent
5. **Validate Data**: Review each patient's responses
   - Check AI-calculated scores
   - Mark as Valid or Invalid
   - Provide corrections if needed
6. **Export Results**: Complete all 350 validations and download PDF report

### PHQ-9 Scoring Guide

| Score Range | Severity Level |
|-------------|----------------|
| 0-4 | Minimal Depression |
| 5-9 | Mild Depression |
| 10-14 | Moderate Depression |
| 15-19 | Moderately Severe Depression |
| 20-27 | Severe Depression |

**Response Values:**
- Not at all = 0
- Several days = 1
- More than half the days = 2
- Nearly every day = 3
